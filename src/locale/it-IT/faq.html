Frequently-Asked Questions:

Miglioro davvero la mia sicurezza o risparmio solo qualche sforzo?

Dipende. Password Hasher può aumentare la sicurezza di molti utenti liberandoli dal peso dell'utilizzo di password complicate. D'altra parte se siete estremamente disciplinati dovreste essere consapevoli che memorizzando molte password di siti potreste essere più vulnerabili se la Master Key vi fosse sottratta. Un eventuale hacker potrebbe avere successo solo se sapesse usare il programma per ottenere le password generate. Se utilizzi spesso la medesima password questa estensione ti aiuterà.

Che accadrebbe se l'estensione non fosse disponibile e volessi loggarmi ai miei siti?

Puoi generare una pagina web da caricare e lanciare in ogni browser quando non è disponibile questa estensione. Puoi anche usare l'apposito online tool per generare le stesse password, ma questo tool non può sapere l'id del sito e i settaggi. Consulta l'aiuto Pagina portatile per maggiori informazioni..

La pagina portabile generata si adatta perfettamente alla sidebar di firefox per un più facile utilizzo. Tutto il necessario è racchiuso in una sola pagina, così puoi salvare il file HTML in altri dispositivi di archiviazione, come chiavi USB.

Le opzioni avanzate di filtraggio e di lunghezza rendono questa estensione incompatibile con altri programmi similari, es. Hashapass, che può generare risultati differenti, a seconda del settaggio che stai usando.

Devo salvarmi le password generate da qualche parte?

Questa non è una cattiva idea, ma salvando un pagina portatile come backup hai tutto, ma non la/le Master Key. Sta a te decidere. Una ulteriore precauzione come una copia su carta o un salvataggio ulteriore è una misura extra di sicurezza, nel caso tu abbia bisogno di reinserire manualmente la password generate . Puoi anche usare un programma come KeePass (molto raccomandato) per salvare i tuoi dati in file cifrati.

Pensate a tutti gli scenari di computer morenti o presi in prestito o di programmi difettosi. :)

Devo ricordarmi le opzioni per ogni sito?

Normalmente no. Il programma li ricorda per te. Ma se perdi i dati salvati, per esempio eliminando le password di firefox, devi ricreare impostazioni di default e reimpostarle per ogni sito. E' auspicabile che non abbia tante occasioni per ricordarti di questo problema.

Dove sono conservati i dati?

Tutto è conservato nella cache delle password di Firefox, codificato e al sicuro. Puoi usare estensioni come Password Exporter per salvarti questi dati in un backup o per trasferirli in un altro dispositivo.

Come posso ottenere aiuto, riferire un problema o fare un suggerimento?

Puoi liberamente contattare lo sviluppatore tramite email. Questa non è ovviamente una grande organizzazione, ma vi risponderò il più velocemente possibile e apprezzerò ogni vostro feedback. (in English, please:)
