Introduzione

Come ottenere una buona sicurezza:

  1. Password complesse sono più difficili da scoprire.
  2. Usare password diverse per ogni sito.
  3. Cambiare periodicamente le proprie password.

Perché probabilmente tu non sei abbastanza sicuro:

  1. Password complesse sono difficili da ricordare.
  2. Manipolare una gran quantità di password può diventare problematico.
  3. Aggiornare le password comporta problemi nel rimemorizzarle.

Come Password Hasher può aiutarti:

  • Le password complesse vengono generate automaticamente.
  • La stessa Master Key genera differenti password per ogni sito.
  • Puoi aggiornare velocemente le password tramite il tasto varia (es: google------>google:1----->google:2 ecc..).
  • Puoi aggiornare la Master Key senza dover aggiornare allo stesso modo tutti i siti.
  • Supporta differenti lunghezze per le password.
  • Supporta la punteggiatura e le lettere maiuscole.
  • Tutti i dati vengono salvati nel database di sicurezza del browser.

Concetti fondamentali:

Id sito

L' id sito è semplicemente un nome, per esempio "google" o "msn", assegnato al sito, e usato per generare la Master Key. Per livelli di sicurezza leggeri e medi viene assegnato automaticamente un id ad ogni sito, che viene inserito ogni volta vi si torni. L' id del sito deve essere unico. Non occorre che sia complesso.

Master Key

La Master Key è una password complessa che scegli tu ed usi per uno o più siti. Una volta che l'id del sito si è combinato con la Master Key per creare un'unica password, può darsi che tu voglia un'altra password per una tua ulteriore sicurezza. A questo punto, un sito alla volta devi iniziare ad aggiornare le password basate sulla nuova Master Key. Non devi quindi aggiornare tutti i siti nella stessa maniera.

Password generata

La password generata è il risultato della combinazione tra la Master Key e l'id del sito. Essa diviene la password del sito. Devi solo fornire la Master Key a Password Hasher e la password generata verrà inserita nella casella di testo del sito.

Utilizzo di Password Hasher

E' possibile accedere a password hasher in uno dei seguenti modi:

  • Cliccando sul simbolo visualizzato accanto alla casella di inserimento della password.
  • Premendo la combinazione di tasti Control-F6 quando ci si è posizionati sul campo password.
  • Scegliendo Password Hasher sia dal menù strumenti sia cliccando col tasto destro del mouse una volta posizionati sul campo della password.
