Einleitung

Vorgehensweise zur Gewährleistung hoher Sicherheit:

  1. Starke Passwörter, die schwer zu erraten sind.
  2. Verschiedene Passwörter für jede Seite.
  3. Von Zeit zu Zeit bestehende Passwörter ändern.

Warum Sie wahrscheinlich keine hohe Sicherheit haben:

  1. Starke Passwörter sind nur schwer zu merken.
  2. Der Umgang mit einer Vielzahl an Passwörtern ist eine Qual.
  3. Passwörter zu aktualisieren trägt zum Merkproblem bei.

Wie Password Hasher hilft:

  • Starke Passwörter werden automatisch generiert.
  • Der gleiche Master Key liefert auf vielen Seiten verschiedene Passwörter.
  • Sie können Passwörter schnell aktualisieren, wenn sie die Seitenkennung "variieren".
  • Sie können den Master Key aktualisieren, ohne alle Seiten gleichzeitig anzupassen.
  • Er unterstützt unterschiedlich lange Passwörter.
  • Er unterstützt besondere Anforderungen, z.B. Ziffern und Satzzeichen.
  • Alle Daten werden in der sicheren Passwort-Datenbank des Browsers gespeichert.

Konzepte

Seitenkennung

Eine Seitenkennung ist ein einfacher Name, z.B. "google" oder "msn", der einer Seite zugewiesen und verwendet wird, um einen Master Key zu erstellen. Auf niedriger und mittlerer Sicherheitsstufe wird eine standardisierte Seitenkennung zugewiesen und immer bei Ihrer Rückkehr zu dieser Seite ausgegeben. Diese Seitenkennung muss lediglich einmalig sein, nicht komplex.

Master Key

Ein Master Key ist ein starkes Passwort, das Sie erstellen und für viele (oder alle) Seiten verwenden. Da Seitenkennungen den Master-Key verändern, um einmalige Passwörter zu erstellen, brauchen Sie nur dann mehr als einen Master Key, wenn Sie ihn für zusätzliche Sicherheit aktualisieren wollen. In diesem Fall beginnen Sie damit, Ihre Passwörter basierend auf dem neuen Master Key Seite für Seite anzupassen. Sie müssen nicht alle auf einmal ändern!

Hash-Wort

Ein Hash-Wort ist das Ergebnis einer Vermischung des Master Keys mit der Seitenkennung. Es wird zum Passwort der Seite. Sie geben den Master Key im Password Hasher ein und dieser schreibt das Hash-Wort in das Passwortfeld der Seite.

Den Password Hasher verwenden

Sie können den Dialog auf eine von folgenden Arten aufrufen:

  • Klicken Sie auf den Marker beim Passwortfeld.
  • Drücken Sie die Tastenkombination Strg-F6 innerhalb eines Passwortfeldes.
  • Wählen Sie Password Hasher entweder aus dem 'Extras'-Menü oder per Rechtsklick-Popup-Menü auf einem Passwortfeld.
