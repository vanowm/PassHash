Gradi di sicurezza

  • Basso massimizza i vantaggi provando a ricavare l'id sito, ricordando i dati immessi e rivelando le password nascoste dagli asterischi. Mette al riparo dagli hacker in rete, ma non al sicuro dalle persone che hanno accesso fisico al tuo computer o al browser.
  • Medio aumenta la protezione nascondendo la password generata e obbligando ogni volta a inserire la Master Key. Questa è l'impostazione standard.
  • Alto massimizza la sicurezza, non ricordando nulla e nascondendo tutto. Occorre inserire ogni volta l'id sito e la Master Key e svelare manualmente i campi per vederne il contenuto.
  • Personalizzato permette di specificare impostazioni di sicurezza personalizzate. Consiglio - scegliere leggero, medio a alto prima di personalizzato per iniziare con una ragionevole quantità di opzioni attivate.

Opzioni di visualizzazione

  • Mostra tasto password se selezionato mostra un piccolo simbolo accanto ai campi delle password per un accesso immediato al dialogo di Password Hasher.
  • Mostra tasto rivela password se selezionato mostra un piccolo simbolo accanto ai campi delle password per uno svelamento immediato (rimozione degli asterischi) delle stesse.

Attributi generali

Questi settaggi variano le opzioni standard per assicurare che tutte le password generate soddisfino particolari requisiti:

  • Utilizzare almeno una cifra numerica obbliga la password generata a contenere almeno una cifra numerica.
  • Utilizzare almeno un carattere di punteggiatura obbliga la password generata a contenere almeno un carattere di punteggiatura.
  • Utilizzare minuscole e maiuscole obbliga la password generata a contenere almeno una lettera maiuscola ed una minuscola.

Dimensione predefinita

Questa opzione permette di scegliere la lunghezza standard della password generata (numero di caratteri).

Crea password html

Questo tasto permette di creare una singola pagina html portabile per generare parole criptate quando questa estensione non è disponibile. Viene chiesto dove salvare il file che è poi caricato in una scheda del browser.
