Campi e pulsanti

  • Id sito è il nome univoco assegnato al sito.
  • Master Key è una password complessa di vostra scelta.
  • Password generata è la password generata dalla combinazione della Master Key con l' id sito.
  • Varia aggiunge un numero all'id sito in modo da generare differenti password combinate.
  • Mostra mostra i campi non visualizzati in chiaro (elimina gli asterischi).
  • Opzioni mostra scelte aggiuntive che hanno effetto sulla creazione della password generata.
  • Aiuto visualizza l'aiuto per il dialogo corrente.
  • Annulla abbandona il dialogo corrente senza inserire la password generata nella pagina web.
  • OK chiude il dialogo e immette la password generata nel campo password della pagina web.

Attributi

  • Utilizzare almeno una cifra numerica obbliga la password generata a contenere almeno una cifra numerica.
  • Utilizzare almeno un carattere di punteggiatura obbliga la password generata a contenere almeno un carattere di punteggiatura
  • Utilizzare minuscole e maiuscole obbliga la password generata a contenere sia lettere minuscole che maiuscole.
  • Non caratteri specialinon consentire caratteri speciali nella password generata
  • Solo cifra numerica forze la password generata a contenere solo cifre

Dimensione

Scegli una lunghezza personalizzata per la password generata nel sito corrente.

Navigazione da tastiera

  • Invio ti porta al campo vuoto successivo, chiude la finestra una volta completate le operazioni e inserisce nel sito web la password generata quando è pronta. Questo è generalmente l'unico tasto di navigazione di cui potete aver bisogno, e dovrebbe essere di gran lunga più facile che utilizzare il mouse.
  • Esc chiude immediatamente la finestra di dialogo senza immettere nel sito la password generata.
