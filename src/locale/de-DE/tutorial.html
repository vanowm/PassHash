Einstellungen anpassen:

  • Wählen Sie niedrige Sicherheit, wenn Ihre Maschine anderen Benutzern nicht zugänglich ist.
  • Wählen Sie mittlere Sicherheit für den Normalgebrauch.
  • Wählen Sie Hoch für maximale Sicherheit - auf Kosten von zusätzlichen Eingaben, NICHT aber zusätzlichem Merkaufwand.
  • Aktivieren Sie Passwort-Feldmarker, falls Sie Marker in der Nähe des Passwort-Eingabefeldes für einfacheren Zugang zum Password Hasher-Dialog sehen wollen.

Das Passwort einer Seite ändern:

  1. Wechseln Sie auf den Bereich einer Seite, der Ihnen das Ändern des Passworts erlaubt.
  2. Wählen Sie jedes der (beiden) Felder für das neue Passwort aus.
  3. Rufen Sie den Passwort Hasher auf eine der folgenden Weisen auf:
    • Klicken Sie auf den Marker, falls verfügbar.
    • Drücken Sie die Tastenkombination Strg-F6.
    • Wählen Sie Password Hasher aus dem Rechtsklick-Popup-Menü.
    • Wählen Sie Password Hasher aus dem Browser-Menü 'Extras'.
  4. Übernehmen Sie die automatisch erzeugte Seitenkennung oder geben Sie eine eigene an.
  5. Drücken Sie Enter. Der Mauszeiger wechselt zum Master Key-Eingabefeld.
  6. Geben Sie den Master Key ein.
  7. Drücken Sie Enter. Der Dialog schließt sich und fügt das Hash-Wort in das Passwort-Eingabefeld ein.
  8. Übernehmen Sie die Änderungen, nachdem alle Passwort-Felder ausgefüllt wurden.
  9. Beim nächsten Einloggen gehen Sie ähnlich vor, um das Passwort auf der Login-Seite einzugeben.
  10. Auf Seiten, für die Sie noch alte Passwörter verwenden, ignorieren Sie den Password Hasher einfach.
